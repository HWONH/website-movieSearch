# Vue3 Template with Webpack

## Reference

- [https://github.com/ParkYoungWoong/vue3-webpack-template](https://github.com/ParkYoungWoong/vue3-webpack-template)

## Installation

```bash
$ npx degit HWONH/vue.js-webpack-template-basic DIRECTORY_NAME
$ cd DIRECTORY_NAME
$ npm i
$ npm run dev
```
